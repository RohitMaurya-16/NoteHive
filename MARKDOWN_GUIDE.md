# NoteHive Markdown Guide

Welcome to NoteHive! This guide will help you understand how to format your notes using Markdown. Our editor supports a variety of styles to keep your knowledge organized and beautiful.

## 📝 Text Formatting

- **Bold**: Use `**text**` to make text **bold**.
- *Italic*: Use `*text*` or `_text_` to make text *italic*.
- `Inline Code`: Use backticks `` `code` `` to highlight code or technical terms inline.

## 🏗️ Structure

### Headings
Use `#` symbols at the start of a line to create headers (H1 to H4):
- `# Heading 1`
- `## Heading 2`
- `### Heading 3`
- `#### Heading 4`

### Lists
**Unordered List:**
- Use `-` or `*` followed by a space.
- Like this list!

**Ordered List:**
1. Use `1.` followed by a space.
2. The numbers will increment automatically.

### Horizontal Rule
Use `---` on a new line to create a thematic break:

---

## 💡 Advanced Features

### Blockquotes
Use `>` to highlight a quote or important note:
> "Knowledge is power, but sharing it is the ultimate superpower." — NoteHive Team

### Code Blocks
Use triple backticks to create a code block. You can optionally specify a language for labeling:

```javascript
function greet() {
  console.log("Welcome to NoteHive!");
}
```

### Links
Create links using the `[Text](URL)` format:
[Visit NoteHive](https://notehive-notes.netlify.app/)

## 🚀 Writing Tips
- Use **Write Mode** (FiEdit) to edit your notes.
- Use **Read Mode** (FiEye) to see the beautiful rendered output.
- Click the **Preview** button for a full-screen formatted view.
